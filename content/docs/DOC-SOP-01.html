1 Purpose & Scope

This procedure ensures all controlled documents and records are uniquely identified, reviewed and approved before use, available in their current version at the point of use, protected against loss and unintended change, and retained for the required period.

Scope: every document and record required by the QMS, including external documents. Operating model is electronic-only (no paper originals); the access-controlled folder holds the released version and the system version history is the audit trail. Satisfies ISO 13485 4.2.4/4.2.5, ISO 27001 7.5 and ISO 42001 7.5.

3.1 Identify & number

InputNeed for a new or changed document
Activity• Record the need for the document in the Document List (DOC-LIS-04).
• Assign a unique ID as DOMAIN-CLASS-NN per the naming convention (DOC-LIS-05).
• Select and clone the correct master template (SOP, deliverable or list).
• Set the initial version number to 01.
ResponsibleQMR (R)
Method / CriteriaNaming per DOC-LIS-05; one ID per document
Tools / TemplateDOC-LIS-04; DOC-TPL-01/02/03
Output / RecordRegister entry
ClauseISO 13485 4.2.4 c)

3.2 Draft / revise

InputRegistered document; master template
Activity• Draft or revise the document using tracked changes.
• Apply the layout and structure rules (header band, sections, footer).
• Confirm the content is correct, complete and consistent with the rest of the QMS.
• Update the change description for the version.
ResponsibleAuthor (R)
Method / CriteriaTracked changes; master template used
Tools / TemplateDOC-TPL-01/03
Output / RecordDraft
ClauseISO 13485 4.2.4 a)

3.3 Review

InputDraft
Activity• An independent reviewer (not the author) checks content correctness, regulatory fit, layout and spelling.
• Verify that changes are traceable and that dependent documents are unaffected or are updated accordingly.
• Use the SOP quality checklist (DOC-LIS-01) for SOPs.
• Return the document to the author if the review is not passed.
ResponsibleQMR / Reviewer (R)
Method / CriteriaDual control; reviewer independent of author
Tools / TemplateDOC-LIS-01 / DOC-FOR-01
Output / RecordReviewed draft
ClauseISO 13485 4.2.4 a)

InputReviewed draft
Activity• Approve the document for use; Manual and Policy approved by Top Management.
