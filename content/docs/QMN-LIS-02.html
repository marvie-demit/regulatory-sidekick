Device Applicability Matrix

How to use this register
PurposeScope the QMS to YOUR device. The QMS is one Core (always on) plus switch-on modules. This tool tells you which modules, standards and documents apply to your specific device.
Step 1Open the 'Device Profile' sheet. For each characteristic, mark Applies? = Y or N for your device.
Step 2For every Y, the 'Then activate' column tells you which module(s) to switch on. Read the 'Applicability Matrix' sheet for the standards and the exact documents.
Step 3Activate those modules/documents; record any non-applicable characteristics as justified exclusions in the QMS Manual.
ComposableA device can match several characteristics at once (e.g. active + implantable + software + AI). Activate ALL that apply - the modules compose. There is no 'deferred' tier: there is Core (always on) + optional modules selected by device qualification/type.
Optional modulesEvery module here is OPTIONAL - you activate it only if your device has the characteristic; modules you do not need simply stay dormant for your device (they are not deleted).
