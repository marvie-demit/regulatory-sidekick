1 Purpose & Scope

This procedure ensures top management reviews the management system at planned intervals for continuing suitability, adequacy and effectiveness, on the basis of analysed data, and that improvement actions result.

Scope: the whole quality management system. It satisfies ISO 13485 5.6 & 8.4, ISO 27001 9.3 & 9.1 and ISO 42001 9.3.

2 Roles & Responsibilities

RoleResponsibility

Guidance (delete before release): Each step is a structured configuration (IDEF0/ICOM): Input, Activity, Responsible (RACI), Method/Criteria (Control), Tools/Template (Mechanism), Output/Record, Clause.

3.1 Data collection & analysis

InputProcess data across the QMS
Activity• Collect data on quality, complaints, CAPA, audits, PMS, risk, suppliers, and security & AI performance and incidents.
• Analyse the data using appropriate (statistical) methods to determine performance against objectives and to identify trends.
• Flag signals that require management attention.
ResponsibleQMR (R); Process Owners (Support)
Method / CriteriaISO 13485 8.4; against defined KPIs/objectives
Tools / TemplateRegisters / dashboards
Output / RecordData analysis
ClauseISO 13485 8.4 / 27001 9.1

3.2 Prepare review inputs

InputAnalysed data
Activity• Compile all required review inputs (audit results, feedback/complaints, process & product conformity, CAPA status, follow-up from prior reviews, changes, recommendations).
• Add security and AI performance, risk status and regulatory changes.
• Prepare the agenda and circulate the input pack.
ResponsibleQMR (R)
Method / CriteriaInputs per ISO 13485 5.6.2 + 27001/42001
Tools / TemplateReview input pack
Output / RecordReview input pack
ClauseISO 13485 5.6.2

3.3 Conduct management review

InputReview input pack
Activity• Top Management reviews the suitability, adequacy and effectiveness of the QMS across all standards.
• Discuss each input and the need for changes, resources and improvements.
• Record the minutes.
ResponsibleTop Management (R)
Method / CriteriaPlanned interval (at least annually)
Tools / TemplateMRV-TPL-01
Output / RecordReview minutes
ClauseISO 13485 5.6.1 / 27001 9.3 / 42001 9.3

3.4 Decide outputs & track

InputReview minutes
Activity• Decide on improvements to the product and QMS, resource needs and changes to objectives.
• Assign actions with owners and due dates.
• Track actions to completion via CAPA where applicable.
ResponsibleTop Management (R); QMR (Support)
Method / CriteriaActions tracked via CAPA
Tools / TemplateMRV-TPL-01; CAP-SOP-01
Output / RecordManagement Review Report; actions
ClauseISO 13485 5.6.3
