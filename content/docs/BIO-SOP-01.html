1 Purpose & Scope

This procedure ensures the biological safety of any device or part in direct or indirect contact with the body, through a documented biological evaluation conducted WITHIN the risk management process per ISO 10993-1.

Scope: patient-contacting / implantable devices and materials. DORMANT for non-contacting software / IVD-instrument-only products. It satisfies MDR GSPR 10 and the ISO 10993 series, using chemical characterization and toxicological risk assessment to minimise animal testing.

3 Procedure

3.1 Categorize device by nature & duration of contact

InputDevice design; materials list; intended use
Activity• Classify each body-contacting component by contact type (surface / external-communicating / implant).
• Classify contact duration (limited <=24h / prolonged >24h-30d / long-term >30d).
• Confirm the applicable ISO 10993-1 endpoint set for each category.
ResponsibleBiological Safety Lead (R)
Method / CriteriaISO 10993-1 Annex A categorisation
Tools / TemplateBIO-FOR-01 Biological Endpoint Matrix
Output / RecordContact categorisation & endpoint set
ClauseISO 10993-1 / MDR GSPR 10

3.2 Establish the Biological Evaluation Plan (BEP)

InputCategorisation; existing & equivalent-material data
Activity• Identify the required endpoints and the strategy to address each (existing data, chemical characterisation, or testing).
• Perform a data gap analysis against existing/equivalent-material data.
• Plan to minimise animal testing - use characterisation and existing data first.
ResponsibleBiological Safety Lead (R)
Method / CriteriaISO 10993-1 §6 (evaluation within RM)
Tools / TemplateBIO-TPL-01 Biological Evaluation Plan
Output / RecordApproved BEP
ClauseISO 10993-1 §6

3.3 Material & chemical characterization

InputBEP; materials, suppliers, processing aids
Activity• Compile material constituents, suppliers and processing aids.
• Conduct chemical characterisation (extractables / leachables) where the strategy requires it.
• Record constituents in the materials list.
ResponsibleDevelopment / Lab (R)
Method / CriteriaISO 10993-18
Tools / TemplateBIO-LIS-01 Materials & Chemical Characterization List
Output / RecordChemical characterisation data
ClauseISO 10993-18

3.4 Toxicological risk assessment

InputChemical characterisation data
Activity• Derive tolerable intake / allowable limits for identified chemicals.
• Assess the toxicological risk of extractables/leachables against those limits.
ResponsibleToxicologist (R)
Method / CriteriaISO 10993-17
Tools / Template-
Output / RecordToxicological risk assessment
ClauseISO 10993-17

3.5 Biological testing for remaining endpoints

InputGap analysis (endpoints not closed by data/characterisation)
Activity• Conduct biological tests for the open endpoints (cytotoxicity, sensitisation, irritation, etc.).
• Use GLP / accredited laboratories; record results against acceptance criteria.
ResponsibleBiological Safety Lead (R)
Method / CriteriaISO 10993-5/-10/-11/-23 as applicable
Tools / Template-
Output / RecordTest reports
ClauseISO 10993 series

3.6 Biological Evaluation Report & conclusion

InputCharacterisation, toxicological assessment, test data
Activity• Compile the Biological Evaluation Report with the overall biological-safety conclusion.
• Feed residual biological risks into the risk file and the benefit-risk analysis.
ResponsibleBiological Safety Lead (R)
Method / CriteriaISO 10993-1 §6.1
Tools / TemplateBIO-TPL-02 Biological Evaluation Report
Output / RecordBiological Evaluation Report
ClauseISO 10993-1 / MDR GSPR 10

3.7 Re-evaluate on change

InputMaterial / process / sterilisation / supplier / use change
Activity• Assess any such change for biological impact.
• Re-evaluate via change control and update the BER.
ResponsibleBiological Safety Lead (R)
Method / CriteriaPer CHG-SOP-01
Tools / TemplateCHG-SOP-01
