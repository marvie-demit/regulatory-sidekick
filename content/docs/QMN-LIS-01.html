Medical Device Stepwise Implementation Tracker

How to use this register
Model4 phases x 12 Quality System Essentials, adapted from the WHO LQSI tool; end-point = MDR/IVDR certification.
Waves & parallelWithin a phase, activities share a 'Wave'. Same wave => can run in parallel. Use 'Depends on' to see what must finish first and 'Leads to' for follow-ups.
Each activityHas a Statement, Why, How (sub-steps), Who, Documents - matching the detailed guide QMN-MAN-03.
How to useFilter by Phase or Wave to plan. For each activity open its documents, complete them, set Owner/Status/Target date.
StatusNot started / In progress / Done / N-A (reason in Notes).
