1 How to read this guide

Medical Device Stepwise Implementation (MDSI) is split into 4 phases (adapted from WHO LQSI). Within each phase, activities are grouped into 'waves': activities in the same wave can run IN PARALLEL; a later wave depends on earlier ones. Each activity lists its dependencies (what must be done first) so you can see the follow-up chain. Each activity is structured as: Statement, Why, How (chronological sub-steps), Who, Documents, Depends on. Track status in QMN-LIS-01.
